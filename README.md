# Vencord Fork with nsfwGateBypass Plugin

A custom Vencord fork including the recently removed nsfwGateBypass plugin.

## Requirements
- Node.js
- PNPM package manager

## Installation
```bash
git clone https://github.com/kauht/venkord
```

```bash
cd venkord
```

```bash
pnpm inject
```
